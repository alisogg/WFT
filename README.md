# 🌸 Kintsugi

> *"El kintsugi es el arte japonés de reparar cerámica rota con oro, resaltando las grietas en lugar de esconderlas. Cada mujer en STEM que rompe una barrera deja una línea dorada para las que vienen después."*

**Kintsugi** es un repositorio de recursos gratuitos y curados para mujeres y niñas interesadas en STEM (Ciencia, Tecnología, Ingeniería y Matemáticas). Aquí encontrarás cursos, guías vocacionales, becas y comunidades para empezar, continuar o reinventar tu camino en STEM.

## 🌱 Cómo contribuir

Este repo crece gracias a la comunidad. Si conoces un curso, beca o recurso que debería estar aquí:

1. Haz un **fork** del repo
2. Agrega el recurso en la sección correspondiente siguiendo el formato existente
3. Verifica que el link funcione y sea gratuito (o indícalo claramente si no lo es)
4. Abre un **Pull Request** describiendo qué agregaste

También puedes abrir un [Issue](../../issues) si prefieres solo sugerir un recurso sin subir el PR tú misma.
